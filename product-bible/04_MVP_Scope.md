# Objective

This document defines the minimum set of features required to launch Kharcha.

A feature should only be included if it helps users:

- Understand their spending
- Control their budget
- Build a daily tracking habit

Everything else is postponed.

# Launch Goal

A user should be able to:

- Register
- Add expenses
- Track budget
- View reports
- Read Money Story

Nothing else is required for launch.

---

# Launch Definition

Kharcha V1 launches when:

- All MVP features work correctly.
- The application is stable.
- Expense entry takes less than 5 seconds.
- Money Story is functional.
